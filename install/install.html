Установка

Скачайте скрипт инсталлятора: правой кнопкой мыши - сохранить как...

Запустите его для установки необходимых пререквизитов.

После установки внизу экрана появится сообщение: 

  Prerequisites have been installed, please run again
- это означает, что требуемые программы установлены и скрипт нужно запустить второй раз для установки самой программы и запуска настройки.

После установки файлов программы появиться диалоговое окно, где надо указать имя сервера asterisk, внутренний номер оператора и его пароль. Эти данные можно получить у администратора системы.

Если внизу окна появится сообщение 

  Error, run again
- свяжитесь с разработчиком

Обновление

Для обновления программы необходимо ещё раз запустить установочный скрипт. При выполенении обновления появится окно ввода параметров подключения, его можно просто закрыть, чтобы сохранить уже имеющиеся настройки.
